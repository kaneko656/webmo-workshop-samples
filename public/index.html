ポインタを水平・北向きに設置します。

北向きに自動で合わせる方法は次の通りです。

  1. ポインタを水平・北向きに設置します。
  2. 調整ボタンを押します。
自動でWebmoの方向を調整する
  1. ポインターを水平にします。
  2. ZIG simulator アプリを起動し、Settingsを開きます。
  3. IP ADDRESS: 「WebmoのIP（Webmoのディスプレイで確認できます）」を入力
    PORT NUMBER: 「5555」を入力
    MESSAGE RATE: 「1」にする
  4. Sonsorを開き、「COMPASS」にチェックを入れ、Startを開きます。
  5. ポインターとスマホの方向を合わせます。スマホの置き場所はどこでも大丈夫です。
  6. 自動調整ボタンを押すと、自動で北向きに回転します。
